# Longest Common Prefix

## Overview
This project solves the **Longest Common Prefix** problem from LeetCode.  
The objective is to find the longest prefix string that is shared among an array of strings.

This challenge focuses on string manipulation, edge case handling, and efficient iteration.

---

## Problem Description
Given an array of strings, return the longest common prefix among them.  
If there is no common prefix, return an empty string.

### Example
```text
Input: ["flower","flow","flight"]
Output: "fl"
